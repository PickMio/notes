怎样才能成熟起来？

满意答案

姑娘，此话怎 3级 2009-06-26 

1、遇到不想回答的问题，直视对方的眼睛，微笑、沉默。 

2、走路抬头挺胸，心情不好时，不想跟人招呼，点头微笑，径直走过。 

3、请记得，好朋友的定义是：你混的好，他打心眼里为你开心；你混的不好，他由衷的为你着急。  

4、做自己的决定。然后准备好承担后果。从一开始就提醒自己，世上没有后悔药吃，而我永远有个plan B。 

5、自己分内的事情，努力做到一百分。 

6、接受自己不过是个“小小的我”，但眼里要能够悦纳“大大的世界”。 

7、If you love something, set it free, if it comes back to you, it is yours, if it doesn't , it never was. 

8、不要试图给自己找任何借口，错误面前没人爱听那些借口。

9、不要随意发脾气，谁都不欠你的。 

10、不说谎话，因为总有被拆穿的一天。 

11、别低估任何人。 

12、你没那么多观众，别那么累。 

13、过去的事情可以不忘记，但一定要放下。 

14、别人说的记在脑袋里，而自己的，则放在心里。 

15、社会是有等级的，很多事不公平。别抱怨，因为没有用。 

16、你永远没有你自己想象中那么重要

17、钱能解决的问题统统不叫问题。 

18、无论何时说“我爱你”，请真心实意。无论何时说“对不起”，请看着对方的眼睛。 

19、永远不要以貌取人。慢慢地说，但要迅速地想。 。 

20、找点时间，单独呆会儿。 

21、不是自己的东西不要，就是再喜欢也不行，要懂得放弃。 

22、不要觉得是生活亏欠了你，其实是我做的努力不够。 

23、努力向前，再努力向前，再努力一下下，愿望就会实现

做到你就成熟了

热心问友 2009-06-26 

20岁左右的人，介于童年和成年之间的年龄阶段，是个想变得成熟可总是甩不掉幼稚的年纪，人生的路是一步步走出来的，而成熟又是个很抽象的概念，不能笼统的定义说谁谁变成熟了，只能说他在某些事情的处理能力变得老练的，即使成年人，在对待一些事情上，也有犯幼稚的时候。我也不过才22岁，有些时候对自己表现出来的不成熟和愚蠢感到羞愧，责问自己为什么不够成熟。环境可以改变人得心态，在社会上待久了，人自然就会变得成熟和老道，人生的经验是一点点累积出来的，一夜暴富也许可能，一下子变成熟是万万不能的。你要端正自己的人生观价值观，要懂得明辨是非，明白忠孝仁义礼智信的涵义，给自己的人生树立好道德标准。人可以圆滑，但不可以奸诈，做人要做到外圆内方才是成功的人生。

David 13级 2009-06-29 

该说话的时候说，不要多说讲重点，不要多余的表情；动作举止适可而止，不要太夸张；穿着不要太幼稚，这些都是表面的。但看上去让人就觉得比较成熟，想让自己都认为自己成熟那就注意自己的思想了。其实这些东西都是在生活中慢慢积累起来的，多一点生活阅历，多一点生活经验，理智的看待事物。外表上的成熟只能靠穿衣打扮，内在的成熟需要时间的磨练，经历过才会懂得，就算你没有经历过，也可以在长辈们身上学习人生道理。很多关于做人处世的书籍值得一看，你花点时间在书本上。在职场上，要改变在学生时代的那些骄气和不好习惯，工作上要做个认真负责，有担当做事的人。在生活中，自己的经历就是最好的素材让自己成熟，就算长辈如何教导自己，当自己真正面对问题的时候，才会深刻领会到个中人生道理。人生的态度、价值观也会慢慢改变，思想也会成熟。成熟的人，会多方面多角度去思考和分析自己的路。
